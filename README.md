**Hello world!**
This is my GIS project portfolio consisting of the latest spatial data studies I've done. I'm currently working on a Kyiv Public Transit Assessment project, where I built a Static GTFS for the entirety of Kyiv city with limited and fragmented data availabilty. I'm planning to update this folder to include the 9-step Python workflow using pandas, ArcPy and TkinterMapView GUI I used create realistic routing system for private public transit simulation based on cleaned and processed OSM topology Network Dataset.

**Kyiv city atlas project** - a comprehensive geospatial study on Kyiv, Ukraine, exploring the location within the country and the metropolitan area, demography and concentration of people within districts and subdivisions, land use and city's complex transportation network.
